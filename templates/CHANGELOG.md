# Changelog

All notable changes to this project will be documented in this file.

---

## [1.1] - 2026-01-26

### Added
- Main menu landing page
- Error Log HL7 Parser
- Single HL7 Message Parser
- Accession and CCL accession display
- OBR and OBX numbering (OBR1, OBX2, etc.)
- Copy buttons for accession, OBR, and OBX codes
- Per-message collapse/expand with visual indicators
- Error and warning banners scoped to correct HL7 message
- Highlighting of OBX referenced in error text
- Handling of unmatched / unsolicited orders
- Consistent UI styling across tools
- Application branding and favicon
- Authorship and license metadata

### Fixed
- Error message bleed across HL7 messages
- Incorrect association of errors to unrelated messages
- HL7 parsing issues caused by mixed metadata and payloads

### Notes
- This version establishes a stable foundation for future enhancements.
